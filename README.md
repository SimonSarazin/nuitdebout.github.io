## Faire tourner le site sur sa machine

- Avec PHP

```
$ cd nuitdebout.github.io
$ php -S localhost:8000

```

- Avec Node.JS

```
$ cd nuitdebout.github.io
$ npm install -g http-server
$ http-server
Starting up http-server, serving ./
Available on:
  http://127.0.0.1:8080
  http://192.168.1.81:8080
  http://10.10.10.1:8080
  http://192.168.0.1:8080
```
